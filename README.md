# Sales Analysis Dashboard Using Excel

## Project Description
This project is a Sales Analysis Dashboard developed using Microsoft Excel. The dashboard helps analyze sales performance by calculating Total Sales, Average Sales, Minimum and Maximum Sales, and identifying high-performing orders. It also categorizes cities as Metro and Non-Metro and calculates commission rates and commission amounts based on sales performance.

## Objectives
- Analyze sales data efficiently
- Calculate total and average sales
- Identify highest and lowest sales values
- Classify sales performance using IF conditions
- Categorize cities using logical functions
- Calculate commission rates and commission amounts

## Features
- Total Sales Calculation
- Grand Total of Sales
- Average Sales Analysis
- COUNT and COUNTA Functions
- Minimum and Maximum Sales Identification
- Performance Classification (High/Low)
- City Category Classification (Metro/Non-Metro)
- Commission Rate Calculation
- Commission Amount Calculation

## Tools Used
- Microsoft Excel
- Excel Formulas (SUM, AVERAGE, COUNT, COUNTA, MIN, MAX, IF, Nested IF, OR)

## Dataset
Sample Sales Dataset used for practicing Excel formulas and data analysis concepts.
